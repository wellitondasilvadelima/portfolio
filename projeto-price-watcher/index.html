PROJETO: OBSERVADOR DE PREÇOS

A automação Observador de Preços (PriceWatcher) tem como objetivo monitorar os preços de produtos em um e-commerce com base em uma lista de links fornecida. Sempre que um preço de um produto diminuir, a aplicação informará o usuário sobre a alteração através de uma planiha de preços, ajudando a acompanhar descontos e promoções de forma eficiente.

DESCRIÇÃO DO PROJETO

O propósito da aplicação é facilitar o processo de monitoramento de preços, permitindo que o usuário acompanhe as flutuações de preços de múltiplos produtos sem precisar visitá-los manualmente. Isso é útil, por exemplo, para quem deseja comprar produtos quando o preço atinge um valor mais baixo.

PriceWatcher é uma aplicação automatizada desenvolvida para monitorar os preços de produtos em e-commerces de maneira prática e eficiente. Através da leitura de links de produtos armazenados em uma planilha Excel, a aplicação realiza o scraping dos preços atuais e os compara com os preços anteriores. Quando um produto tem seu preço alterado (diminuído ou aumentado), a aplicação atualiza a planilha com o status da alteração e fornece informações detalhadas sobre cada produto.

TECNOLOGIAS UTILIZADAS NO PROJETO

O PriceWatcher é uma aplicação desktop desenvolvida em python, atualmente o E-commerce ao qual a automação verifica os produtos é a KaBuM! um comércio eletrônico brasileiro de tecnologia, especializado em hardware, linha gamer, eletrônicos, smart home e smartphones.

O Python foi a escolha principal pela sua flexibilidade e robustez, permitindo o desenvolvimento ágil e eficiente da aplicação. Para realizar requisições HTTP e acessar as páginas dos produtos, utilizamos Requests, que facilita a comunicação com os sites e a coleta das informações. O BeautifulSoup entra em cena para realizar o scraping das páginas, extraindo de forma precisa os preços dos produtos. Para manipulação e análise dos dados coletados, o Pandas é essencial, oferecendo funcionalidades poderosas para comparar os preços e gerar relatórios completos em Excel.

Para garantir a perfeita interação com os arquivos Excel, contamos com openpyxl, que nos permite ler e gravar dados de forma eficiente. Além disso, a aplicação roda de forma automatizada e em segundo plano, graças ao agendador de tarefas do windows é possível que a aplicação faça o monitoramento sem necessidade de intervenção manual, basta agendar horarios para iniciar a aplicação.

SOBRE O SOFTWARE

Com o PriceWatcher, o monitoramento de preços de produtos nunca foi tão fácil e eficiente! A automação elimina a necessidade de verificar manualmente os preços de diversos itens, poupando seu tempo e esforço. O sistema ajudará o usuário sobre alteração de preço — seja uma queda, aumento ou estabilidade — permitindo que você tome decisões de compra mais informadas e no momento certo. Além disso, a aplicação funciona de forma autônoma em segundo plano, atualizando os dados, sem a necessidade de sua intervenção constante, bastando apenas informar os produtos que deseja monitorar. Com o PriceWatcher, você fica sempre por dentro das melhores ofertas e descontos.
